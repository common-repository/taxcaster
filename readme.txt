== Description ==

This plugin embeds the TaxCaster Lite Tax Refund Estimator application on any post or page in WordPress

Estimate your 2011 tax refund. How big will your tax refund be? This application makes it easy to find out how much you’re going to get
back from Uncle Sam this year or how much you’ll owe.

[View the full TaxCaster Calculator](http://turbotax.intuit.com/tax-tools/calculators/taxcaster/ "TaxCaster Calculate 2011 Tax Return")

== Installation ==

1. Upload the TaxCaster folder to the plugins directory in your WordPress installation
2. Activate the plugin
3. Add [taxcaster] to any post or page to view the TaxCaster application
3. Navigate to the TaxCaster menu for help and plugin settings

That's it!
